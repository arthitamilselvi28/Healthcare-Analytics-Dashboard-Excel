# Healthcare-Analytics-Dashboard-Excel

## Project Overview
This project presents a Healthcare Analytics Dashboard created using Excel to analyze hospital data and visualize key healthcare metrics. The dashboard helps in understanding patient trends, appointments, and overall hospital performance.

## Features
- Interactive healthcare dashboard
- Patient and appointment analysis
- Visual charts and summaries
- Easy-to-understand healthcare insights

## Tools Used
- Microsoft Excel

## Project File
```
Healthcare_Analytics_Dashboard.xlsx
```

## Purpose of the Project
The goal of this project is to demonstrate basic healthcare data analysis and dashboard creation using Excel.
